### [ Note ]
1. WSL(Windows Subsystem for Linux)을 사용하면 QGroundControl 에서 오류가 발생하는 것을 확인하였습니다.(2024-06-20)

   Ubuntu Desktop 환경을 이용해야 합니다.
   
2. 실행 시 항상 패키지 update와 upgrade를 진행합니다.

   ```bash
   sudo apt update
   sudo apt upgrade
   ```
